# 불편 사항

VSCode GitHub Pull Request
확장(`vscode-pull-request-github-local-patches`)에는
github pull request: changes in pull requests 라는 사이드바가 있다. 이
사이드바를 이용하면 코드 리뷰를 편하게 할 수 있는데, VSCode 워크스페이스를
환경에서 여러 폴더가 열려 있는 경우 이 사이드바에서 해당 PR이 열려있지 않기
때문에 불편하다.

# 핵심 목표

VSCode 워크스페이스 폴더를 위에서 언급한 사이드바에서 편리하게 리뷰할 수 있도록
만든다.
